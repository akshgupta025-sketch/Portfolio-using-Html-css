# Personal Portfolio Website

A responsive personal portfolio website built using **HTML** and **CSS**. This project showcases my profile, skills, and social links while helping me learn the fundamentals of front-end web development.

## 🚀 Features

- Responsive design for different screen sizes
- Clean and modern UI
- Social media links
- Hover effects and animations
- Profile image section
- Mobile-friendly layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Font Awesome Icons
- Google Fonts
